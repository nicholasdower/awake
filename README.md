# awake

## Install

```shell
brew install nicholasdower/tap/awake
```

## Uninstall

```shell
brew uninstall awake
```

## Help

```
usage: awake [-d] [<duration>]

Description

    Keep your Mac awake, optionally for the specified duration (e.g. 12h30m).

Options

    -d, --daemonize  Daemonize.
    -h, --help       Print help.
    -v, --version    Print version.
```
