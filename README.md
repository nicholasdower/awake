# awake

## Install

```shell
brew install nicholasdower/tap/awake
```

## Uninstall

```shell
brew uninstall awake
```

## Help

```
usage: awake [<duration>]

Description

    Stay awake, optionally for the specified duration.

Options

    -h, --help       Print help.
    -v, --version    Print version.
```
